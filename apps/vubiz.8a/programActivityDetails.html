This creates an Excel report of a Selected Active Learner or All Active Learners and the Programs they have accessed during the selected dates. Leave date fields empty to return all activity ranges. Leave the User Id field empty to return all active learners. Guest Program accesses are not included. Tap Generate Spreadsheet and the number of rows required will be calculated and displayed. If you'd like to continue, then reports containing 5000 or less rows will be generated in formatted Excel while reports with more than 5000 rows will be generated in unformatted CSV.
